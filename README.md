# Turned based driving game web frontend

[Live demo here!](https://dvf.ovh/)

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```
